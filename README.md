## Développement

### Installation

```
git clone https://github.com/wdjennane/WEB2_P2021_FINAL_PROJECT
npm install
```

### Dev mode

```
npm start
```

### Prod mode

```
npm run build
```

### Lancer le linter

```
npm run lint
```
